# Music Board Visualizer 🎵

音乐木板可视化器 - 小球沿着木板下落，碰撞激活颜色，配合音乐节拍。

## 效果
- 小球从顶部下落，依次碰撞木板
- 碰撞时木板亮起颜色 + 粒子爆炸效果
- 小球沿贝塞尔曲线平滑转向
- 歌词文字随节拍淡入
- 紫蓝色渐变背景 + 光效

## 技术栈
- Three.js + TypeScript + Vite

## 运行
```bash
npm install
npm run dev
```
